### *¿Cómo se puede resolver?* 🌌
Podemos utilizar Google Colab o Jupyter local.

Los pros y contras:

- Colab (pros): no requiere instalación, arranque rápido, fácil compartir.   
- Colab (contras): dependemos de internet y del entorno de Google.
- Jupyter local (pros): control total de nuestro entorno y continuidad para proyectos.
- Jupyter local (contras): requiere tener Python/kernel/librerías bien instalados.
